# Flex Yemen - فلكس يمن

تطبيق Flutter متكامل للتجارة الإلكترونية والمحفظة الإلكترونية في اليمن.

## 🚀 المميزات

### وضع PRO الاحترافي
- 3 مستويات: Lite / PRO / Expert
- 10 أسواق متعددة
- 25 قسم رئيسي
- 30 منتج رائج
- إحصائيات حية
- رسوم بيانية متقدمة

### المحفظة الإلكترونية
- إيداع وسحب
- تحويلات داخلية وخارجية
- دفع الفواتير
- شحن الرصيد
- بطاقات الهدايا

### التصميم
- دعم الوضع الليلي والنهاري
- دعم اللغة العربية (RTL)
- ألوان ذهبية مميزة
- رسوم متحركة سلسة

## 📱 الشاشات

### المصادقة
- تسجيل الدخول
- إنشاء حساب
- نسيت كلمة المرور
- التحقق من OTP

### الصفحة الرئيسية
- 3 أوضاع: Lite / PRO / Expert
- التنقل السفلي
- إجراءات سريعة

### المحفظة
- بطاقة الرصيد
- الإجراءات السريعة
- الخدمات
- سجل المعاملات

### الملف الشخصي
- معلومات المستخدم
- الإحصائيات
- الإعدادات
- تسجيل الخروج

## 🛠️ التقنيات المستخدمة

- Flutter 3.x
- Dart 3.x
- Provider (State Management)
- Shared Preferences
- Flutter Animate
- Google Fonts

## 📦 التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/flexyemen/app.git
cd flex_yemen
```

2. تثبيت الحزم:
```bash
flutter pub get
```

3. تشغيل التطبيق:
```bash
flutter run
```

## 🏗️ بنية المشروع

```
lib/
├── models/          # نماذج البيانات
├── providers/       # مزودات الحالة
├── services/        # الخدمات
├── screens/         # الشاشات
├── widgets/         # الويدجتس المشتركة
├── theme/           # الثيم والألوان
└── main.dart        # نقطة الدخول
```
